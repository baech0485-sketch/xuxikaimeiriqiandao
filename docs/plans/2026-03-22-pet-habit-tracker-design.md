# 小宠伴学 — 儿童习惯养成宠物系统设计文档

## 1. 产品概述

**产品名称**：小宠伴学
**目标用户**：5-8岁小朋友（幼小衔接~小学低年级）
**核心目的**：通过宠物喂养机制，激励小朋友每日完成学习任务，养成良好习惯
**主要设备**：平板（横屏优先），响应式兼容手机

## 2. 核心玩法循环

1. **早上打开** → 宠物在小窝里打招呼，显示今天的6个任务
2. **完成任务** → 点击任务卡片确认完成 → 获得一份食物
3. **喂养宠物** → 把食物拖给宠物 → 宠物开心地吃掉，播放可爱动画
4. **全部完成** → 触发"今日全勤"庆祝动画 + 获得一颗金星
5. **连续打卡** → 积累连续天数 → 宠物越来越开心
6. **忘记打卡** → 宠物变饿变伤心 → 连续3天不喂 → 宠物离家出走

## 3. 宠物状态机制

| 状态 | 条件 | 表现 |
|------|------|------|
| 开心 | 今天已被喂食 | 摇尾巴/蹦跳动画，说"好开心！" |
| 饿了 | 今天还没喂食 | 眼巴巴看着，说"我饿了~" |
| 伤心 | 连续2天没喂 | 流眼泪动画，说"你是不是忘了我..." |
| 出走 | 连续3天没喂 | 宠物消失，显示空窝+留言信 |

**召回机制**：宠物出走后，需要连续3天全勤完成所有任务才能召回。

### 6.2 领养页面

- 5种宠物横向排列，每个宠物有动画预览
- 点击宠物查看性格描述
- 输入宠物昵称
- 确认领养按钮

### 6.3 打卡日历页面

- 月视图日历，标记每天完成情况
- 绿色=全勤，黄色=部分完成，灰色=未完成
- 点击某天可查看当日详细任务完成情况

## 7. 视觉风格

- **色调**：柔和糖果色（浅蓝、浅粉、浅黄、薄荷绿）
- **卡片**：大圆角（16-24px），柔和阴影
- **按钮**：最小48px触控区域，胖胖的圆角按钮
- **字体**：大号友好字体，任务文字至少18px
- **动画**：Framer Motion 实现宠物呼吸、摇尾巴、吃东西、星星飞溅等效果
- **反馈**：完成任务时播放星星飞溅动效

### API端点

- `GET/POST /api/user` — 获取/创建用户
- `GET/POST /api/daily` — 获取/更新今日打卡
- `POST /api/pet/feed` — 喂养宠物
- `POST /api/pet/status` — 更新宠物状态
